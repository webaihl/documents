## 递归函数

> 使用递归实现整型数组的求和

+ 解决步骤
  1. 拆分成子问题
  2. 找到临界条件(用于停止递归)

+ 代码实现

  ~~~java
  //l代表数组的左边界的下标
  private int sum(int[] arr, int l) {
      if(l == arr.length) return 0;
      int x = sum(arr, l+1);
      return arr[l] + x; // arr[l] + sum(arr, l+1)
  }
  ~~~
